# Little Milestones

**AI-powered monthly growth reports for babies and pets**

Little Milestones 是一个用 AI 生成成长月报的小工具。

它可以把日常照片描述、事件记录或结构化 JSON 数据，整理成一份温暖、可保存的月度成长总结，适用于宝宝和宠物等成长记录场景。

这个项目不想先做一个复杂的相册系统，而是先把“值得保存的成长纪念”这件事做好。

它关注的是：

- 从零散记录中提炼出动作、表情、成长变化和高光时刻
- 将素材整理成自然、有温度的月报内容
- 帮助用户把日常碎片变成真正值得留存的成长纪念

## 项目简介

在日常记录里，很多珍贵的成长瞬间往往分散在照片、随手备注和事件清单中，不容易被系统地保存下来。

Little Milestones 希望用更轻量的方式，把这些零散输入整理成一份完整的月报，让“记录成长”这件事更简单，也更有纪念意义。

## 适用场景

- 宝宝成长记录
- 宠物成长记录
- 每月家庭纪念整理
